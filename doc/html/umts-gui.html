NAME

umts-gui.pl - A graphic user interface for GSM/UMTS terminals

SYNOPSIS

umts-gui.pl [options]

DESCRIPTION

umts-gui.pl is the graphic user interface of the umts-tools package. Most of its functionalities work well, but it should still be considered as experimental as it is relatively young compared to the other umts-tools scripts.

umts-gui.pl makes use of Gtk2::Perl, so you will need to make sure that Gtk2::Perl is installed in order to run the script.

OPTIONS

Common options: -d debugging mode -l<log> write log to <log> -h display this help message -p<port> terminal is connected to serial port <port> e.g. '\\.\COM12', '/dev/usb/acm/0' -z use dummy terminal
