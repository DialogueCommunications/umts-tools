NAME

umts-answer.pl - A GSM/UMTS call answering tool

SYNOPSIS

umts-answer.pl [options]

DESCRIPTION

The umts-answer.pl script allows you to automatically answer mobile-terminated phone calls.

OPTIONS

 Common options:
  -d          debugging mode
  -l<log>     write log to <log>
  -h          display this help message
  -p<port>    terminal is connected to serial port <port>
              e.g. '\\.\COM12', '/dev/usb/acm/0'
  -z          use dummy terminal
