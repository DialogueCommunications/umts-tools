NAME

umts-p2klog.pl - A data log parser for Motorola P2k handsets

SYNOPSIS

umts-p2klog.pl [options]

DESCRIPTION

The umts-p2klog.pl script allows you parse binary logs captured on Motorola P2k handsets with a tool such as datalog-tool provided by flash-tools.

flash-tools can be downloaded from http://flash-tools.jerryweb.org/.

OPTIONS

 Options:
  -h          display a help message
  -i<input>   read input from <input> instead of standard input
