NAME

umts-phonebook.pl - A GSM/UMTS terminal phonebook tool

SYNOPSIS

umts-phonebook.pl [options]

DESCRIPTION

The umts-phonebook.pl script allows you to read entries from a handset's phonebook or to write entries to it. On most terminals, this gives you access to both the SIM phonebook and the terminal's own phonebook. This allows you to make backups of your phonebook to plaintext files and restore them at a later date.

OPTIONS

 Common options:
  -d          debugging mode
  -l<log>     write log to <log>
  -h          display this help message
  -p<port>    terminal is connected to serial port <port>
              e.g. '\\.\COM12', '/dev/usb/acm/0'
  -z          use dummy terminal
  
 Options:
  -b<book>    operate on phonebook <book> from terminal
              examples : SM (SIM), ME (terminal)
  -f<file>    read/write phonebook to <file>
  -s          write to the selected phonebook in terminal
  -x          extract selected phonebook from terminal
